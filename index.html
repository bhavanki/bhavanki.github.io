About Me

I'm a software developer by trade, working for the fine folks at Cloudera since mid-2013, but with fifteen years of professional experience before that. I've always tried to share code and libraries I've made with others, but pre-Cloudera it was difficult for me to do so in a public repository due to contractual restrictions. Nowadays, though, I really enjoy using GitHub as a place to offer up what I've got and grab what others provide.

These days I work on Cloudera Director. It's not open source, but pieces of it are!

  • Director uses plugins for supporting cloud providers. There are an SPI and associated TCK available for you to start with.
  • The plugins that Director uses for AWS, Google Cloud Platform, and Azure are all open source. We also have an example plugin that can be a useful starting point.
  • Lots of other goodies are available in the director-scripts repo.
  • While we don't accept PRs for it since it's automatically generated, our SDK code is available for use.

Before working on Director, I was active in the following Apache projects.

  • Apache Accumulo [Github mirror] - committer and PMC member
  • Apache ZooKeeper [Github mirror] - contributor

Feel free to check out any of those projects (pun intended).

For more about me, check my LinkedIn profile.

Repositories

I'll highlight some of my own public repos. They are all small projects. My goal is to get something mentioned on One Thing Well. :)

See them all on my repo page.

moving-hadoop-to-the-cloud

Oh, hey, I wrote a book on Hadoop in the cloud! This repository holds the code from the book, and perhaps a little bit more ...

lasa

You don't have root, so you can't set up monitoring on servers where your code lives. This little utility gets you something, at least: a simple report on a variety of things you can still check on without having root access. You can even set up email notifications if something goes wrong.

litlca

OpenSSL incantations are obscure. Use this set of scripts to more easily run a CA using openssl. Great for testing.

javacodeformatter

I'm not beholden to an IDE. Still, there are times when you need to format your Java code, say, according to your team's standards, and you don't want to spin up Eclipse just to do it. This JAR takes care of things from the command line.

simplenote-export

These scripts help massage plain text export of Simplenote notes for use in other systems. I created them to get my notes ready for access via Notational Velocity.

titubate

This is a framework for running "randomwalk" tests, which means your test runner wanders randomly across a directed graph of test actions. The framework is refactored out of Apache Accumulo.
